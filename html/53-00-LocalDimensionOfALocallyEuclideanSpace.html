local dimension of a locally Euclidean space


Let X be a locally Euclidean space. Recall that the local dimension of X in y∈X is a natural number n∈ℕ such that there is an open neighbourhood U⊆X of y homeomorphic to ℝn. This number is well defined (please, see parent object for more details) and we will denote it by dimy⁢X.

Proposition. Function f:X→ℕ defined by f⁢(y)=dimy⁢X is continuous (where on ℕ we have discrete topology).

Proof. It is enough to show that preimage of a point is open. Assume that n∈ℕ and y∈X is such that f⁢(y)=n. Then there is an open neighbourhood U⊆X of y such that U is homeomorphic to ℝn. Obviously for any x∈U we have that U is an open neighbourhood of x homeomorphic to ℝn. Therefore f⁢(x)=n, so U⊆f-1⁢(n). Thus (since y was arbitrary) we’ve shown that around every point in f-1⁢(n) there is an open neighbourhood of that point contained in f-1⁢(n). This shows that f-1⁢(n) is open, which completes the proof. □

Corollary. Assume that X is a connected, locally Euclidean space. Then local dimension is constant, i.e. there exists natural number n∈ℕ such that for any y∈X we have

dimy⁢X=n.

Proof. Consider the mapping f:X→ℕ such that f⁢(y)=dimy⁢X. Proposition shows that f is continuous. Therefore f⁢(X) is connected, because X is. But ℕ has discrete topology, so there are no other connected subsets then points. Thus there is n∈ℕ such that f⁢(X)={n}, which completes the proof. □

Remark. Generally, local dimension need not be constant. For example consider X1,X2⊆ℝ3 such that

X1={(x,0,0)|x∈ℝ}  X2={(x,y,1)|x,y∈ℝ}.

One can easily show that X=X1∪X2 (with topology inherited from ℝ3) is locally Euclidean, but dim(0,0,0)⁢X=1 and dim(1,1,1)⁢X=2.
