integral curve


Definition Suppose M is a smooth manifold, and X is a smooth vector field on M. Then an integral curve of X through a point x∈M is a curve c:I→M, such that

c′⁢(t) = (X∘c)⁢(t),for all t in I
c⁢(0) = x.

Here I⊂ℝ is some open interval of 0, and c′⁢(t) is the tangent vector in Tc⁢(t)⁢M represented by the curve.

Suppose xi are local coordinates for M, ci are functions representing c in these local coordinates, and X=Xi⁢∂∂⁡xi. Then the condition on c is

d⁢cid⁢t⁢(t)=Xi∘c⁢(t),for all t.
