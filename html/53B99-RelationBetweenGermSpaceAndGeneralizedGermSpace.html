relation between germ space and generalized germ space


Let X, Y be a topological spaces and x∈X. Consider the germ space (http://planetmath.org/GermSpace) and the generalized germ space (http://planetmath.org/GermSpace) at x:

Gx⁢(X,Y);Gx*⁢(X,Y).

If f:X→Y is a continuous function, then we have an induced element [f]∈Gx*⁢(X,Y). It can be easily seen, that if [f]=[g]∈Gx⁢(X,Y), then [f]=[g]∈Gx*⁢(X,Y). In particular we have a well-defined mapping

τ:Gx⁢(X,Y)→Gx*⁢(X,Y);
τ⁢([f])=[f].

Proposition 1. τ is injective.

Proof. Indeed, assume that τ⁢([f])=τ⁢([g]) for some f,g:X→Y. Let f′:U→Y and g′:U′→Y be a representatives of τ⁢([f]) and τ⁢([g]) respectively. It follows, that there exists an open neighbourhood V⊆X of x such that

f|V=f|V′=g|V′=g|V.

In particular [f]=[g] in Gx⁢(X,Y), which completes the proof. □

Proposition 2. If X is a normal space and Y is a normal absolute retract (for example Y=ℝ), then τ is onto.

Proof. Assume that [f]∈Gx*⁢(X,Y) for some f:U→Y. Since X is regular (because it is normal) then there exists an open neighbourhood V⊆X such that the closure V¯⊆U. Now since X is normal and Y is a normal absolute retract, then f|V¯ can be extended to entire X (by the generalized Tietze extension theorem). It is easily seen that any such extension gives the same element in Gx⁢(X,Y) (and it is independent on the choice of the representative f) and if F:X→Y is an extension of f|V¯, then

τ⁢([F])=[f]

because F|V=f|V. This completes the proof. □

Remark. If in addition Y is a topological ring (for example Y=ℝ), then it can be easily checked that τ preserves ring structures. In particular if X is normal and Y=ℝ or Y=ℂ, then τ is an isomorphism of rings. Also it is a good question whether the assumptions in proposition 2 can be weakened.
